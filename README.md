# Great 12 Weeks (위대한 12주)

12주 동안 핵심 목표 3가지를 달성하기 위한 목표 관리 웹 어플리케이션입니다.

## 🚀 시작하는 방법 (How to Start)

개발자가 아니셔도 걱정하지 마세요! 아래 단계만 따라하시면 바로 시작할 수 있습니다.

### 1. 프로그램 실행하기
터미널(명령어 창)에서 아래 명령어를 입력하고 엔터(Enter)를 치세요:

```bash
npm run dev
```

### 2. 웹사이트 열기
명령어를 입력하면 잠시 후 아래와 같은 글자가 나옵니다:

```
  ➜  Local:   http://localhost:5173/
```

이제 인터넷 브라우저(크롬 등)를 켜고 주소창에 `http://localhost:5173` 을 입력하시면 앱이 실행됩니다.

## ✨ 주요 기능
- **12주 목표 설정**: 가장 중요한 3가지 목표를 정하세요.
- **주간 액션 실행**: 매주 목표를 달성하기 위해 해야 할 일들을 체크하세요.
- **85% 달성**: 매주 85% 이상 달성하여 승리하는 한 주를 만드세요.
- **대시보드**: 12주간의 여정을 한눈에 확인하세요.

## ⚠️ 데이터 저장
모든 데이터는 현재 사용 중인 브라우저에 저장됩니다. 창을 닫아도 데이터는 유지되지만, 브라우저 캐시를 삭제하면 데이터가 날아갈 수 있으니 주의하세요.
